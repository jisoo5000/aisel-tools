## 네이밍 규칙

구조: [고유명] + [디테일(있을 경우)] + [카테고리명]

### 회피 기준 (3가지 동시 충족 시에만 회피)
- 같은 고유명 + 같은 카테고리 + 같은 핏 → 불가
- 하나라도 다르면 사용 가능

예시: SIE "아방 와이드 트라우저" 있어도
- 에이슬 "아방 부츠컷 팬츠" → 가능 (핏 다름)
- 에이슬 "아방 타이 블라우스" → 가능 (카테고리 다름)
- 에이슬 "아방 와이드 팬츠" → 불가 (3가지 전부 겹침)
