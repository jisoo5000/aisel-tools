# AISEL 네이밍 규칙 (2026.07.24 기준)

---

## 1. 네이밍 구조

**[고유명] + [핵심 디테일(있을 경우)] + [카테고리명]**

- 디테일 없으면: `고유명 + 카테고리명`
- 디테일 있으면: `고유명 + 디테일 1-2개 + 카테고리명`

예시:
- 베아 셔츠
- 베아 스트라이프 셔츠
- 루아 슬리브리스 맥시 드레스

---

## 2. 고유명 선정 기준

유럽계 여성 인명 또는 프랑스 지명/조어, 2-3음절.

### 회피 기준 — 아래 3가지 **동시 충족** 시에만 회피 (하나라도 다르면 사용 가능)

| 기준 | 내용 |
|------|------|
| 같은 고유명 | SIE 또는 에이슬 기존에 동일 이름 있음 |
| 같은 카테고리 | 바지>바지, 셔츠>셔츠 등 |
| 같은 핏 | 와이드>와이드, 부츠컷>부츠컷 등 |

### 판단 예시

| SIE 기존 제품 | 에이슬 후보 | 가능 여부 | 이유 |
|---|---|---|---|
| 아방 와이드 트라우저 | 아방 와이드 팬츠 | ❌ 불가 | 3가지 전부 겹침 |
| 아방 와이드 트라우저 | 아방 부츠컷 팬츠 | ✅ 가능 | 핏 다름 |
| 아방 와이드 트라우저 | 아방 타이 블라우스 | ✅ 가능 | 카테고리 다름 |
| 헤일리 리브드 니트 스커트 | 헤일리 컴포트팬츠 | ✅ 가능 | 카테고리 다름 |
| 루아 린넨 숄 가디건 | 루아 시어 셔츠 | ✅ 가능 | 카테고리 다름 |
| 루아 린넨 숄 가디건 | 루아 와이드 팬츠 | ✅ 가능 | 카테고리 다름 |

## 9. 네이밍 요청 처리 순서 (클로드 기준)

1. SIE 해당 카테고리 실시간 fetch
2. 에이슬 기존 확정 고유명 전체와 겹침 확인
3. 회피 기준 3가지 동시 충족 여부 판단
4. 후보 10개 이상 제시
