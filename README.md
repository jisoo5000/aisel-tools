# AISEL Tools

에이슬(AISEL) 패션 이커머스 운영 자동화 도구 모음. 디자인 회의용 PPT 생성 스크립트(`make_ppt_final.js`, `ppt-maker` 저장소)와 원가 계산기·작업지시서(`cost-calculator.html`, `work-order.html`)의 기준 데이터를 문서화합니다.

---

## 📊 에이슬 원가 기준표

카테고리별 원가 구조 (단위: 원). `제조원가 = 원단비 + 공임 + 시야게 + 부자재 + 로스분` 이며, 목표원가는 판가의 약 25% 수준으로 잡습니다.

| 코드 | 카테고리 | 판가 | 목표원가 | 원단단가(/y) | 요척(y) | 원단비 | 공임 | 시야게 | 부자재 | 로스분 | 제조원가 | 부자재 상세 |
|------|----------|------:|--------:|------------:|-------:|------:|-----:|------:|------:|------:|--------:|-------------|
| `dm_top` | 다이마루 상의 | 59,000 | 14,000 | 5,385 | 1.3 | 7,000 | 6,000 | 0 | 0 | 1,000 | 14,000 | – |
| `jk_ts` | 직기 티셔츠 | 59,000 | 14,000 | 4,385 | 1.3 | 5,700 | 6,000 | 1,300 | 0 | 1,000 | 14,000 | – |
| `bl` | 블라우스 | 69,000 | 17,250 | 4,344 | 1.6 | 6,950 | 8,000 | 1,300 | 0 | 1,000 | 17,250 | – |
| `sh` | 셔츠 | 69,000 | 17,000 | 3,938 | 1.6 | 6,300 | 8,000 | 1,300 | 400 | 1,000 | 17,000 | 단추×10 = 400 |
| `cp` | 컴포트 팬츠 | 59,900 | 14,900 | 4,207 | 1.45 | 6,100 | 6,000 | 1,300 | 500 | 1,000 | 14,900 | 밴드 500 |
| `csk` | 컴포트 스커트 | 59,900 | 14,900 | 5,083 | 1.2 | 6,100 | 6,000 | 1,300 | 500 | 1,000 | 14,900 | 밴드 500 |
| `jk_pt` | 직기 팬츠 | 69,000 | 17,250 | 4,027 | 1.85 | 7,450 | 7,000 | 1,300 | 500 | 1,000 | 17,250 | 밴드 500 |
| `jk_sk` | 직기 스커트 | 59,900 | 14,900 | 4,875 | 1.2 | 5,850 | 6,000 | 1,300 | 750 | 1,000 | 14,900 | 밴드+지퍼 750 |
| `op` | 원피스 | 98,000 | 24,500 | 7,975 | 2.0 | 15,950 | 6,000 | 1,300 | 250 | 1,000 | 24,500 | 지퍼 250 |

- **원단비** = `원단단가 × 요척` (반올림). 예: 다이마루 상의 `5,385/y × 1.3y ≈ 7,000`
- **로스분**은 전 카테고리 공통 고정 1,000원
- **시야게**(마무리 가공)는 다이마루 상의만 0원

## 🖼️ 슬라이드 레이아웃 정보

`make_ppt_final.js`가 생성하는 디자인 회의용 PPT의 슬라이드 한 장 구성 (레이아웃: **16:9**, 10 × 5.625 inch).

### 좌측 — 제품 이미지
- 위치 `x:0, y:0`, 크기 `3.2 × 5.625 inch`, `cover` 방식으로 꽉 채움
- 카카오톡 원본 사진 폴더에서 파일명 자동 매칭(`.png`/`.jpg`/`.jpeg`)

### 우측 상단 — 헤더
- 슬라이드 번호 (우측 끝, 8pt, 회색)
- 제품명 (12pt Bold, `Apple SD Gothic Neo`)
- 목표원가 라인 (10pt, 액센트 블루 `#1D4ED8`)

### 우측 — 원가표
헤더 배경 `#2F5496` / 흰 글씨. 4개 컬럼 · 6개 행.

| 항목 | 예상 금액 | 실제 금액 | 비고 |
|------|-----------|-----------|------|
| 원단비 | `원단단가/y × 요척y = 원단비` (**11pt**) | (기입란) | |
| 공임 | 금액 | | |
| 시야게 | 금액 또는 — | | |
| 부자재 | 상세 = 금액 (있을 때 9pt) | | |
| 로스분 | 금액 (고정) | | |
| **제조원가** | 금액 (Bold, 액센트) | | |

- 폰트: 기본 11pt, 부자재 상세만 9pt, 원단비 값은 명시적으로 **11pt** (`size: 11`)
- 제조원가 행은 강조 배경(`#EEF2FF`) + 볼드 + 액센트 컬러

### 우측 하단 — 발주표
헤더 `발 주` (`#444444`). 컬럼: `컬러 / S / M / L / XL / 소계 / 비고`, 데이터 행 `컬러 1`, `컬러 2` (기입란).

### 최하단 — 성과표
헤더 `성 과` (`#666666`). 행: `예상 판매량`, `실제 성과` (기입란).

### 컬러 팔레트
| 용도 | HEX |
|------|-----|
| 헤더 배경 | `#2F5496` |
| 액센트 | `#1D4ED8` |
| 제조원가 배경 | `#EEF2FF` |
| 구분선 | `#DDDDDD` |
| 흐린 텍스트 | `#999999` |
